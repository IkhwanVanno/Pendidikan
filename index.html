CODE CLASS

Front End Dev

Pengembang Front-End adalah seseorang yang membuat situs web dan aplikasi web. Perbedaan antara Front-End dan Back-End adalah Front-End merujuk pada tampilan halaman web, sedangkan back-end merujuk pada cara kerjanya. Anda dapat menganggap Front-End sebagai sisi klien dan Back-End sebagai sisi server . Bahasa dasar untuk Pengembangan Front-End adalah HTML, CSS, dan JavaScript.

Back End Dev

Pengembang back-end adalah ahli yang membangun dan memelihara mekanisme yang memproses data dan melakukan tindakan di situs web. Tidak seperti pengembang front-end, yang mengontrol segala sesuatu yang dapat Anda lihat di situs web, pengembang back-end terlibat dalam penyimpanan data, keamanan, dan fungsi sisi server lainnya yang tidak dapat Anda lihat.
